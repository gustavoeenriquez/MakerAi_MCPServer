# MakerAi MCP Server

> ⚠️ **Este proyecto ha sido absorbido por [MakerAI](https://github.com/gustavoeenriquez/MakerAi).**

## ¿Qué pasó?

El servidor MCP para Delphi que comenzó aquí evolucionó y fue integrado
directamente en el framework **MakerAI**, donde se mantiene activamente
junto con el cliente MCP, los agentes, RAG y los demás componentes.
